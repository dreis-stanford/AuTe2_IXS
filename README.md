## 🚀 Quick Start
### Interactive Analysis
```bash
python analyze_q.py
Then enter Q-vectors in primitive coordinates:
Enter Q (primitive): 0.5 0 0
Enter Q (primitive): 0 0 4.1
Programmatic Use
from code.single_q_analysis import SingleQAnalyzer
from code.force_constants import ForceConstants
import numpy as np
# Load crystal structure and force constants
xtal = ForceConstants("data/AuTe_2_m.fc")
Phi = xtal.convert_to_eV_per_Angstrom2()
masses = np.array([xtal.masses[xtal.atom_type_map[i]-1] for i in range(xtal.nat)])
# Create analyzer
kT_THz = 6.2  # Temperature in THz (~300 K)
analyzer = SingleQAnalyzer(xtal, Phi, masses, kT_THz)
# Analyze a Q-point
result = analyzer.analyze([0.5, 0, 0], coords='primitive')
# Access results
print(f"Frequencies: {result['frequencies_cm']} cm⁻¹")
print(f"IXS (Stokes): {result['IXS_stokes']} barn/(unit cell·sr)")
See examples/ directory for more examples.

•Freq: Phonon frequency (wavenumbers or meV)
•L-char: Longitudinal character (0 = transverse, 1 = longitudinal)
•IXS(S): Stokes scattering cross-section (phonon creation)
•IXS(AS): Anti-Stokes cross-section (phonon annihilation)
•Au%, Te1%, Te2%: Atomic participation percentages
⚠️ Important Note on L-char and Participation
The L-character and atomic participation percentages depend on eigenvector phase conventions and may differ from MATLAB. This is expected and does not affect the physics.

🔧 Technical Details
Phonon Calculation
•Uses reduced q in first Brillouin zone
•Solves dynamical matrix: D(q) = Σ Φ(R) e^(-iq·R) / √(m_i m_j)
•Enforces acoustic sum rule
IXS Cross-Section
The differential cross-section per unit cell per steradian:
dσ/dΩ = r_e² |F(Q,ω)|² / ω × (n + 1)  [Stokes]
dσ/dΩ = r_e² |F(Q,ω)|² / ω × n        [Anti-Stokes]
Where:
•r_e = classical electron radius
•F(Q,ω) = structure factor
•n = Bose-Einstein occupation factor
•ω = phonon frequency
Key convention:
•Phonons calculated at reduced q (first BZ)
•Form factors use full |Q| (including reciprocal lattice vector)
•Phase factors use full Q
Structure Factor
F(Q,ω) = Σ_s (f_s / √m_s) × e^(-iQ·r_s) × (Q · e_s)
Where:
•f_s = atomic form factor
•m_s = atomic mass
•r_s = atomic position
•e_s = eigenvector (displacement)

🐛 Troubleshooting
Q: IXS values seem too small
•Check that you're using the full Q (not reduced to first BZ) for form factors
•The code automatically handles this in single_q_analysis.py
Q: L-char differs from MATLAB
•This is expected! See NOTES.md for explanation
•IXS values are still correct
Q: Atomic participation shows wrong atoms
•Eigenvector conventions differ between solvers
•Sum Te1 + Te2 for total Te contribution
